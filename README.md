# Python
Python for beginners

## Python & Data Analysis Learning Journey

Welcome to my personal repository dedicated to mastering Python fundamentals, Object-Oriented Programming (OOP), and Data Analysis with Pandas. 

## About This Repo
This repository serves as a structured, step-by-step archive of my learning progress. It contains organized code files, practical exercises, and documentation tracking my path from coding fundamentals to data-driven analysis.

## What This Repo Covers
The repository is systematically divided into 5 core conceptual phases:
1. **Python Basics & Core Fundamentals:** Data types, operators, operations, and conditional statements.
2. **Data Structures & Collections:** Mastering lists, collections, and deeply understanding dictionaries.
3. **Functions:** Introduction to modular programming and functional practice problem-solving.
4. **Object-Oriented Programming (OOP):** Deep dive into objects, classes, the 4 pillars of OOPs, polymorphism, and data hiding using access specifiers.
5. **Data Analysis (Pandas):** Practical, byte-sized learning modules aimed at mastering the Pandas library for real-world data tasks.

## My Way of Learning & Why I Created This
*   **Structured Progression:** I believe in building a rock-solid foundation. I structured this repo sequentially so that each topic logically builds upon the last (from core types up to advanced OOP and data analysis).
*   **Hands-on Practice:** Instead of just reading theory, I write dedicated practice files (like `function-practice.py`) to test edge cases and enforce muscle memory.
*   **Why I Created This:** I built this workspace to track my personal growth, maintain clean code standards, and create a reusable reference point for future data analysis projects.
